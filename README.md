# Library Management System (C++)

A console-based Library Management System developed using Object-Oriented Programming concepts in C++.

## Features

* Add new books
* Display all books
* Search books by title
* Register library users
* Display registered users
* Borrow books
* Return books
* Save data to text files
* Load saved data automatically

## Technologies Used

* C++
* STL (Vector, String)
* File Handling

## Project Structure

```
Library Management System/

```

## How to Run

### Compile

```bash
g++ main.cpp Library.cpp Book.cpp User.cpp -o library
```

### Execute (Linux/Mac)

```bash
./library
```

### Execute (Windows)

```bash
library.exe
```
